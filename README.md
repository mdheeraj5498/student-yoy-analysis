**Overview**
This project performs a comprehensive year-over-year analysis of student data from MBA and Executive MBA programs. It showcases the full data analysis pipeline — from raw CSVs with inconsistencies to cleaned data, SQL joins, Python transformations, and visualizations in Tableau and Power BI.

**Objectives**

Clean and validate raw student, enrollment, and performance data
Perform relational joins using SQL across multiple tables
Conduct year-over-year comparisons of enrollment and GPA
Visualize trends using interactive dashboards

**Data Pipeline**
**Data Cleaning:**
Removed nulls, duplicates
Fixed formatting issues (year values, program types)
Imputed missing GPA values

**SQL Integration:**
Joined students, enrollments, programs, and performance
Filtered by year and program type (MBA/EMBA)

**Python Analysis:**
Generated KPIs: Enrollments, GPA by year and program
Saved intermediate results for dashboards

**Dashboarding:**
Created slicers, charts, and YoY comparison visuals in Tableau and Power BI
